# 地图笔记

一个纯前端的网页地图标记工具。在地图上点一下就能标记位置、添加标签和备注。支持好友分享、路线规划、高德导航。

## 功能

- 📍 **地点标记**：点地图任意位置添加标记，自定义标签（多选），写备注
- 🏷️ **自动识别**：标记时自动调用高德 API 识别地点名称（小区、学校、商铺等）
- 🔍 **地址搜索**：顶部搜索栏输入地址，实时联想，一键飞到目标位置
- 📋 **标签筛选**：侧边栏按标签筛选标记，多选支持，实时搜索
- 🔗 **路线规划**：多个标记点串联成路线，步行实际路径 + 公交导航跳转
- 👤 **好友分享**：生成分享码，好友导入后能看到你的标记和路线（本地存储，无服务器）
- 🎨 **好友标记**：每个好友自定义颜色，标记叠加显示在地图上
- 📱 **移动端适配**：窄屏左滑面板、右滑菜单，操作不误触地图
- 📤 **分享选中**：勾选部分标记单独分享

## 技术栈

HTML + CSS + JavaScript，单文件 `index.html`，无框架、无构建工具。

- 地图引擎：Leaflet.js
- 底图：高德地图
- 搜索/识别/导航：高德 Web 服务 API
- 数据存储：浏览器 localStorage
- 分享：Base64 编码，纯本地无服务器

## 使用方法

1. 用浏览器打开 `index.html`
2. 点地图添加标记 → 选标签、写备注 → 保存
3. 左侧栏筛选、搜索标记
4. 两个标记以上可以连线规划路线
5. 左侧面板设昵称 → 生成分享码发给好友

## 本地运行

需要 `config.js` 文件（已加入 `.gitignore`），内容为：

```js
var AMAP_KEY = '你的高德地图 Web 服务 Key';
```
